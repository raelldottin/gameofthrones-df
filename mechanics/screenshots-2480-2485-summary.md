## Core Mechanics Confirmed
- Dragon Feeding is instant and meat-perishable per Reign
- Campaign Stage Level Cap can be below current dragon level
- Barracks batch training uses slider up to available capacity minus current troops
- Feed action can be blocked by specific item count (red 0 button state)
- MAX FEED values shown against gold/special currency
- Dragon Growth radial tree shows core and outer nodes with upgradeable levels
- Bottom-ring nodes can remain at 0/15 while stronger early power is stacked top-heavy
- Dragon Troop Capacity % is a dedicated stat in Dragon Growth
