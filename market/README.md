## Daily Capture Fields

Each snapshot should record, when visible:

- observation date, time, and refresh countdown;
- Gold, Campaign Favors, Relic Shards, core balances, and basic resources;
- Resources-tab Gold packs and resource exchanges;
- Relic Shards relics, keys, and core conversions;
- Gold-tab keys, relics, cores, and meat;
- Campaign resources and relic offers;
- remaining purchase limits and unit costs;
- balance and rotation changes from the previous snapshot.

Inventory capture is kept separate from purchase recommendations. A listed offer is not assumed to have been purchased.
