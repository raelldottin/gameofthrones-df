# Gold-Standard Armies Only — June 27, 2026

## Standard

An army only qualifies as gold standard when it has:

1. **3/3 troop affinity**;
2. **one dragon that can safely hold vanguard**;
3. **one early-round stabilizer or damage reducer**;
4. **one damage amplifier or enemy debuffer**;
5. **at least two dragons that can actually finish enemies**;
6. **no dragon placed only because of class**.

The goal is to keep only complete battle engines. Test armies, theoretical armies, and future-unlock armies are excluded from the main list.

## Gold-Standard Armies

### 1. Rhysarion / Vhagar / Malachite — Shieldbearers

**Default orientation:** Rhysarion left / Vhagar vanguard / Malachite right  
**Alternate orientation:** Malachite left / Vhagar vanguard / Rhysarion right

| Requirement | Answer |
|---|---|
| 3/3 affinity | Shieldbearers |
| Safe vanguard | Vhagar |
| Stabilizer | Rhysarion Ebbing Fury |
| Amplifier / mitigation | Malachite Forest's Instinct |
| Finishers | Vhagar + Malachite, with Rhysarion supporting |
| Status | Confirmed default gold standard |

This remains the reference army. It has center survival, early suppression, team support, damage, recovery timing, and full affinity.

### 2. Arrax / Velar / Nyrena — Shieldbearers

| Requirement | Answer |
|---|---|
| 3/3 affinity | Shieldbearers |
| Safe vanguard | Velar |
| Stabilizer | Velar sustain / control shell |
| Amplifier / debuffer | Velar Advantage support + Nyrena debuffs |
| Finishers | Arrax + Nyrena, with Velar Tactical damage support |
| Status | Gold-standard Shieldbearer sustain engine |

This qualifies because it is not a Sentinel quota formation. Velar protects and enables the army, Arrax supplies Physical pressure, and Nyrena supplies the opposite-flank debuff/damage role.

### 3. Arulix / Seasmoke / Shimmer — Cavalry

| Requirement | Answer |
|---|---|
| 3/3 affinity | Cavalry |
| Safe vanguard | Seasmoke |
| Stabilizer | Seasmoke control / buff removal + Arulix Fire-damage reduction |

| Finishers | Seasmoke + Arulix, with Shimmer support |
| Status | Gold-standard Cavalry control engine |

This qualifies because the army has control, damage reduction, enemy-stat suppression, and enough finishing pressure without placing any dragon only for class coverage.

### 4. Jagadrix / Dawnseeker / Crimson — Spearmen

| Requirement | Answer |
|---|---|
| 3/3 affinity | Spearmen |
| Safe vanguard | Dawnseeker |

This qualifies because it preserves Spearmen affinity and gives Crimson a debuff shell. Jagadrix lowers stats that help Fire damage land, Dawnseeker holds center, and Crimson supplies Bloodscale Terror pressure.

### Arulix / Seasmoke / Crimson — Cavalry

## Not Gold Standard Right Now

### Tashix / Syrax / Antares — Archers

This is affinity-safe and has damage, but Syrax is still 1★ and its key team engine is not online yet. Until Syrax unlocks Mindful Synergy or later support habits, this is a development army, not gold standard.

### Tashix / Syrax / Crimson — Archers

Crimson has Archer affinity, but the same Syrax problem remains. This becomes a serious gold-standard candidate after Syrax unlocks team-wide support.

### Seasmoke / Syrax / Antares — Archers

This is too support-heavy right now and depends on Syrax future unlocks.

### Caraxes Archer armies

Caraxes has confirmed Spearmen and Cavalry affinity, not Archer affinity. Do not place Caraxes in Archer gold-standard armies unless a future screenshot confirms Archer affinity.

### Sentinel-spread armies

Any army built mainly to put a Sentinel in every formation fails the standard. The question is whether the army is a complete engine.
