# Gold-Standard Army Candidates — June 27, 2026

## Standard

A candidate only qualifies as a gold-standard army when it has:

1. **3/3 troop affinity**;
2. **one dragon that can safely hold vanguard**;
3. **one early-round stabilizer or damage reducer**;
4. **one damage amplifier or enemy debuffer**;
5. **at least two dragons that can actually finish enemies**;
6. **no dragon placed only because of class**.

The goal is to build the strongest battle engines, not to distribute Sentinels evenly across armies.

## Confirmed Default Gold Standard

### 1. Rhysarion / Vhagar / Malachite — Shieldbearers

**Default orientation:** Rhysarion left / Vhagar vanguard / Malachite right  
**Alternate orientation:** Malachite left / Vhagar vanguard / Rhysarion right

| Requirement | Answer |
|---|---|
| 3/3 affinity | Shieldbearers |
| Safe vanguard | Vhagar |
| Stabilizer | Rhysarion Ebbing Fury |
| Amplifier / mitigation | Malachite Forest's Instinct |
| Finishers | Vhagar + Malachite, with Rhysarion supporting |
| Notes | Default reference engine. All other armies should be judged against this structure. |

This remains the gold-standard army because it has a complete engine: center tank, early suppression, support/mitigation, damage, recovery timing, and full affinity.

## Strong Gold-Standard Candidates

### 2. Arrax / Velar / Nyrena — Shieldbearers

| Requirement | Answer |
|---|---|
| 3/3 affinity | Shieldbearers |
| Safe vanguard | Velar |
| Stabilizer | Velar sustain / Taunt / mitigation package |
| Amplifier / debuffer | Velar Advantage support + Nyrena debuffs |
| Finishers | Arrax + Nyrena / Velar Tactical damage |
| Risk | Arrax becomes fragile during Headlong Into Danger. |

This is a real engine, not a Sentinel quota army. Velar protects and enables Arrax. Nyrena supplies the opposite flank pressure and debuffs.

### 3. Arulix / Seasmoke / Shimmer — Cavalry

| Requirement | Answer |
|---|---|
| 3/3 affinity | Cavalry |
| Safe vanguard | Seasmoke |
| Stabilizer | Seasmoke control / buff removal; Arulix Fire-damage reduction |
| Amplifier / debuffer | Shimmer Strength/Initiative reduction; Arulix Stagger pressure |
| Finishers | Seasmoke + Arulix, with Shimmer support |
| Risk | More control-heavy than kill-heavy. May stall if Seasmoke and Arulix do not finish fast enough. |

This is the best current Seasmoke engine until a Caraxes or Crimson variant proves stronger.

### 4. Tashix / Syrax / Antares — Archers

| Requirement | Answer |
|---|---|

| Safe vanguard | Syrax |

This is the correct Syrax army under the affinity rule. Do not force Caraxes here because Caraxes's confirmed affinities are Spearmen and Cavalry, not Archers.

## Not Gold-Standard Yet

### Archer Caraxes armies

Caraxes should not be placed in a Syrax Archer engine unless a future screenshot confirms Archer affinity.

### Sheepstealer-based armies

Sheepstealer is not discovered in the current repo state, so it cannot be used in current gold-standard armies.

### Sentinel-spread armies

Any army that exists mainly to put a Sentinel in the vanguard fails the standard. The question is not whether every army has a Sentinel. The question is whether every army has a full engine.
