## What This Means

### 1. The Archer shell is now affinity-correct

The biggest positive is that **Tashix / Vhagar / Syrax** is a valid 3/3 Archer formation. This replaces **Caraxes / Vhagar / Syrax**, which only had 2/3 Archer affinity because Caraxes showed No Bonus.

### 2. Full affinity still was not enough for low losses

Even with 3/3 affinity and an Easy clear, the army lost **4,468** troops. That is too high for the no-loss / little-loss goal.

### 3. Syrax was the safest piece

Losses by dragon:

- **Tashix lost 1,926** troops.
- **Vhagar lost 1,887** troops.
- **Syrax lost 655** troops.

Syrax survived best. The bleeding came from Tashix and Vhagar.

### 4. Vhagar did not create low-loss Archer stability by himself

Vhagar can hold center, but this report shows that moving Vhagar into the Archer shell does not automatically create a low-loss army. The army still needs better support, better matchup discipline, or more Syrax/Tashix development.

## Recommendation

Keep **Tashix / Vhagar / Syrax** as the corrected Archer shell, but downgrade it for low-loss farming:

**Valid 3/3 Archer army, not low-loss proven.**

If the goal is no-loss / little-loss farming, use this only when:

1. the target is much weaker;
2. the target is not applying heavy pressure to center/left;
3. you need no-duplicate coverage more than minimum troop losses.

Do not count this as one of the proven low-loss armies yet.
