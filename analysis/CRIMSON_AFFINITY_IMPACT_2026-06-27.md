## Confirmed Positive Troop Affinities

Crimson has three confirmed positive affinities:

1. **Spearmen**
2. **Archers**
3. **Cavalry**

This matters because Crimson can be used in multiple 3/3 engines without breaking the new gold-standard rule.

## Army Impact

### Spearmen

Crimson remains valid in:

**Jagadrix / Dawnseeker / Crimson**

This remains the default Spearmen engine because Jagadrix lowers enemy Instinct/Initiative and Dawnseeker holds vanguard while Crimson supplies Fire damage and control.

### Archers

Crimson can now be tested in Syrax Archer engines without breaking affinity:

**Tashix / Syrax / Crimson**

This may become stronger than **Tashix / Syrax / Antares** if Crimson's Bloodscale Terror output beats Antares's current Fire/Vulnerable pressure. The question is not class balance; the question is which right-flank finisher produces better battle results while preserving 3/3 affinity.

### Cavalry

Crimson can also be tested in Seasmoke Cavalry engines:

**Malachite / Seasmoke / Crimson**

or

**Arulix / Seasmoke / Crimson**

These are valid because they preserve affinity and give Crimson a control/survival shell.

## Updated Recommendation

Do not lock Crimson to only the Spearmen army.

Crimson should be treated as a flexible finisher candidate for:

- Spearmen default engine;
- Archer Syrax engine;
- Cavalry Seasmoke engine.

The next useful tests are:

1. **Tashix / Syrax / Crimson** vs **Tashix / Syrax / Antares**.
2. **Arulix / Seasmoke / Crimson** vs **Arulix / Seasmoke / Shimmer**.
3. **Malachite / Seasmoke / Crimson** as a heavier Cavalry control/damage shell.

Crimson's confirmed triple affinity makes these legal gold-standard candidates. Battle reports should decide the final placement.
