## Confirmed Positive Troop Affinities

Caraxes has two confirmed positive affinities:

1. **Spearmen**
2. **Cavalry**

Archers are not confirmed for Caraxes. Do not place Caraxes into Syrax Archer engines unless a later screenshot confirms Archer affinity.

## Army Impact

### Cavalry

Caraxes can now be tested in Seasmoke Cavalry engines without breaking affinity:

**Arulix / Seasmoke / Caraxes**

or

**Malachite / Seasmoke / Caraxes**

The strongest first test is likely **Arulix / Seasmoke / Caraxes** because Arulix brings control and Fire-damage protection while Caraxes becomes the Fire finisher.

### Spearmen

Caraxes can also be tested in Spearmen engines:

**Jagadrix / Dawnseeker / Caraxes**

or

**Rhysarion / Dawnseeker / Caraxes**

These are legal 3/3 candidates if all three dragons share Spearmen affinity in the selected setup. The main question is whether Caraxes's Infernal Burst output beats Crimson's Bloodscale Terror control and Fire burst.

## Updated Recommendation

Caraxes should no longer be excluded from gold-standard army generation due to unresolved affinities.

Caraxes should be treated as a flexible finisher candidate for:

- Cavalry Seasmoke engines;
- Spearmen Dawnseeker engines.

The next useful tests are:

1. **Arulix / Seasmoke / Caraxes** vs **Arulix / Seasmoke / Shimmer**.
2. **Malachite / Seasmoke / Caraxes** vs **Malachite / Seasmoke / Crimson**.
3. **Jagadrix / Dawnseeker / Caraxes** vs **Jagadrix / Dawnseeker / Crimson**.
4. **Rhysarion / Dawnseeker / Caraxes** only when the main gold-standard Shieldbearer army is not being used.

Caraxes is still 1★, so this is a test priority, not an automatic permanent slot.
