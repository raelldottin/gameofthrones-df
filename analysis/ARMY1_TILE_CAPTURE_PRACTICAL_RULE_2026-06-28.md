# Army 1 Tile Capture Practical Rule

Date: 2026-06-28  
Army: Rhysarion / Vhagar / Malachite  
Troops: Shieldbearers  
Target: Tile Defender Lv. 6 Food, Pinkmaiden (1010, 1332)

## Rule

When Army 1 leaves a defender below roughly 500 troops, especially below 200, send the follow-up hit.

## Evidence

The 07:19:13 UTC report was still a stalemate, but the defender had only 107 / 6,000 troops remaining.

The 07:20:13 UTC report became Victory. Malachite made the only visible attack, killed the last 107 troops, and the Lv. 6 Food territory was claimed.

## Why this matters

The 10-round cap can hide a won fight. A stalemate report can still be a winning state if the defender is nearly empty.

## Dragon lesson

Malachite is the cleanup and siege piece in this shell. Rhysarion still provides late-fight value and siege. Vhagar can be dead and the shell can still claim the tile.

## Vaeldra implication

This is more evidence that Vaeldra should not replace Rhysarion or Malachite in Army 1.
